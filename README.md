## これは何
個人的に業務効率化ツールとして作ったものです。
ターミナルをペインで分割してそれぞれでエディタを起動したり、TODO管理のCLIツールを起動したり、端っこで英単語を表示しておいてこまめに英語の勉強ができるようにしたりしたものです。自分用の備忘録も兼ねて公開します。

## 使い方
### Goの英単語表示CLI
ビルドをして適当な場所において、パスを通してください。
英単語と対応する日本語が書いてあるテキストファイルと、コンフィグファイルが必要です。
go_english_toolの中に入っているenglishフォルダをホームディレクトリ(~)に置くと読み取ってくれます。
英単語とそれに対応する日本語が書いてあるテキストファイルからデフォルトでは5個ランダムに選んで標準出力します。
このテキストファイルは自分で作ってもらっても良いです。テキストファイルの先頭から初めての空白行までは無視するのでそこにコメントを書くことが可能です。

### RustのToDoツール
これもビルドしてパスを通してください。
説明が長くなるので、これの使い方は、
todo_rust_cli/README.mdに記載しました。

## サポートツール
これはpowershellを使うこと前提で$PROFILEに内容を追記してください。これについてはあったら便利なだけでなくても動きます。

## wezterm
下記の方のものをもとに改造しました
https://github.com/KevinSilvester/wezterm-config

変更したのは、起動時にペイン分割状態から始まる点と、ウィンドウが最大状態で始まるようにしています。
これを
~/.config/wezterm
となるように配置すれば読み込んでくれます。
